# Padrão: Gancho com Dado Numérico

## Estrutura

```
[Número impactante] em [prazo].

[Negação 1].
[Negação 2].
[Negação 3].

O que [mudei/fiz/aprendi]?

[Uma palavra ou conceito simples].

[Contexto + desenvolvimento em bullets →]

[Conclusão com insight acionável]

[CTA com pergunta aberta]

#hashtag1 #hashtag2 #hashtag3
```

## Por que funciona

O número no início gera curiosidade imediata. As negações criam tensão ("como então?"). A resposta simples (uma palavra) subverte a expectativa de resposta complexa.

## Exemplos da Sprint 01

**Post #9 — ARR +200%**
> "Escalei ARR +200% em 18 meses. Sem contratar mais vendedores. Sem aumentar CAC. Sem mudar o produto. O que mudei? O sistema."

**Post #13 — R$0 para R$1M**
> "R$0 para R$1M ARR em 8 meses. Fiz isso uma vez. Vou contar o que os dados mostraram."

## Variações

- Número de crescimento: "+200% em X meses"
- Número de prazo: "em X dias/meses"
- Número de escala: "X empresas / X clientes"
- Estatística de mercado: "X% das empresas fazem Y errado"

## Quando usar

- Resultado pessoal verificável (não inventar)
- Contexto B2B/SaaS onde dados têm credibilidade
- Quando o dado é contra-intuitivo ou surpreende
